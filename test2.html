What is Ano? ▶
Ano is a zero-dependency web annotation library that supports text highlights, element pins, freehand drawing, and screen recording.
Does recording work across pages? ▶
Yes! The recorder runs in a separate popup window, so the tab capture continues even when you navigate between pages.
Can I draw while recording? ▶
Absolutely. Once recording starts, the toolbar is freed up. Switch to Draw mode (Alt+D) and annotate live — the tab capture includes everything visible on screen.
